# Tetris

This project is a tetris game made with ReactJS. The project was build solely for the purpose of practicing React Hooks.

## Installation
1. Make sure you have [Node.js](https://nodejs.org/en/) installed.
2. Clone this repository.
3. Once cloned, run:

    ```shell script
    $ cd react-tetris-game
    $ npm install
    $ npm audit fix  # optional
    $ npm start
    ```
4. Enjoy the game.
5. Maybe contribute?
